# INT-000 Requirements

## 機能要件

### FR-001: 備品マスタ管理
- 管理者は備品を登録できる（名称・管理番号・カテゴリ・説明）
- 管理者は備品を編集・削除できる
- 備品一覧を表示できる（名称・管理番号・現在の貸出状況）

### FR-002: QRコード生成
- 各備品に対してQRコードを生成できる
- QRコードには備品IDが埋め込まれる
- 管理者画面からQRコードを印刷用に表示できる

### FR-003: 貸出記録（QRスキャン）
- 従業員はスマホのカメラでQRコードをスキャンできる
- スキャン後、借りる人の名前を入力して貸出を記録できる
- 貸出中の備品をスキャンすると返却処理ができる
- 既に貸出中の備品は、別の人には貸し出せない

### FR-004: 貸出状況一覧
- 現在貸し出し中の備品一覧を表示できる（備品名・借りた人・貸出日時）
- 返却済みを含む全履歴一覧を表示できる

### FR-005: 管理者認証
- 管理者はID・パスワードでログインできる
- 管理者専用画面（備品登録・編集・削除・QR生成）はログインしないとアクセスできない
- ログアウトができる

## 入出力・制約

### 入力
- 備品登録：名称（必須）、管理番号（任意）、カテゴリ（任意）、説明（任意）
- 貸出記録：借りる人の名前（必須）
- 管理者ログイン：ユーザーID・パスワード

### 出力
- 備品一覧画面
- 貸出状況・履歴一覧画面
- QRコード画像

### 依存関係
- Next.js（フロントエンド・API Routes）
- Drizzle ORM + Neon DB（PostgreSQL）
- shadcn/ui（UIコンポーネント）
- QRコードライブラリ（qrcode.react 等）
- カメラスキャンライブラリ（html5-qrcode 等）
