# INT-000 US-001: QRコードスキャンによる貸出

## As a
従業員（一般ユーザー）

## I want
備品に貼られたQRコードをスマホのカメラでスキャンして、自分の名前を入力するだけで貸出を記録したい

## So that
面倒な手続きなしに備品を借りられ、台帳への記入漏れも防げる

## Acceptance Criteria
- [ ] スマホブラウザからカメラを起動してQRコードを読み取れる
- [ ] スキャン後、備品名と「借りる人の名前」入力フォームが表示される
- [ ] 名前を入力して「貸出」ボタンを押すと、貸出が記録される
- [ ] 既に貸出中の備品をスキャンした場合、「貸出中です」と表示されて貸出できない
- [ ] 貸出完了後、完了メッセージが表示される
