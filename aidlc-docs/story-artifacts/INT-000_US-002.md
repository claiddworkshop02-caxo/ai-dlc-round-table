# INT-000 US-002: QRコードスキャンによる返却

## As a
従業員（一般ユーザー）

## I want
借りていた備品のQRコードをスキャンして、ワンタップで返却を記録したい

## So that
返却の手間を最小限にでき、管理者に声をかけなくても返却処理が完結する

## Acceptance Criteria
- [ ] 貸出中の備品のQRコードをスキャンすると、「返却する」ボタンが表示される
- [ ] 「返却する」ボタンを押すと返却が記録される
- [ ] 返却完了後、完了メッセージが表示される
- [ ] 未貸出の備品をスキャンした場合は「現在貸出中ではありません」と表示される
